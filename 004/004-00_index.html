Реалізуйте програму, яка моделює роботу простого кафе замовлення напоїв. Використайте шаблони програмування Singleton та Factory.
        Пояснення:
        1. Створіть клас Cafe, який буде використовувати шаблон Singleton для забезпечення єдиного екземпляру кафе.
        2. Створіть клас Beverage, який буде єдиним класом для опису напоїв у кафе.
        3. Використайте шаблон Factory для створення об'єктів типу Beverage відповідно до замовлення клієнтів.
        4. Реалізуйте можливість додавання нових напоїв до меню кафе.
        5. Створіть декілька об'єктів типу Beverage та реалізуйте деякі операції з ними (наприклад, виведення інформації про напій).
